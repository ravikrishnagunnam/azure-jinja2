Macros

create

Create a new data disk to be used in a virtual machine (see virtual_machine).

Parameters

namestring
storageAccountNamestring, name of an existing storage account (see storage_account) to store the disk
vmStorageAccountContainerNamestring, name of the container in the storage account
sizeOfDiskInGBstring, total size of the disk in GB
dataDiskVhdNamestring, name of the VHD as will be displayed by the OS
luninteger = 0,
createOptionstring = “Empty”

Example

{% import 'azure-jinja2/macros/virtual_machine_data_disk.json.j2' as vm_data_disk %}

   {{ vm_data_disk.create(
         name = "datadisk1",
         storageAccountName = "[concat(uniquestring(resourceGroup().id), 'salinuxvm')]",
         vmStorageAccountContainerName = "vhds,
         sizeOfDiskInGB = "100",
         dataDiskVhdName = "datadisk1"
      )
   }}
