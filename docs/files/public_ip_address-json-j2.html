create

Create a Public IP Address.

Parameters

namestring
apiVersionstring
locationstring
typestring, Public IP allocation method, ”Static” or ”Dynamic”
dnsLabelPrefixstring (optional), prefix of the FQDN associated with the Public IP such as <dnsLabelPrefix>.<location>.cloudapp.azure.com

Example

{% import 'azure-jinja2/macros/public_ip_address.json.j2' as public_ip_address %}

{
  "$schema": "https://schema.management.azure.com/schemas/2015-01-01/deploymentTemplate.json#",
  "contentVersion": "1.0.0.0",
  "resources" : [
    {{ public_ip_address.create(
         name = "myPulicIP",
         apiVersion = "2015-06-15",
         location = "West Europe"
       )
    }}
  ]
}

from_vars

Create a Public IP Address with JSON variables.

Parameters

Namestring = “[variables(‘publicIPAddressName’)]”
apiVersionstring = “[variables(‘apiVersion’)]”
locationstring = “[variables(‘location’)]”
typestring = “[variables(‘publicIPAddressType’)]”, Public IP allocation method, ”Static” or ”Dynamic”
withDNSstring = “false”, set to “true” if you want to use the dnsLabelPrefix parameter
dnsLabelPrefixstring = “[variables(‘dnsLabelPrefix’)]”, prefix of the FQDN associated with the Public IP such as <dnsLabelPrefix>.<location>.cloudapp.azure.com

Example

{% import 'azure-jinja2/macros/public_ip_address.json.j2' as public_ip_address %}

{
  "$schema": "https://schema.management.azure.com/schemas/2015-01-01/deploymentTemplate.json#",
  "contentVersion": "1.0.0.0",
  "variables": {
    "apiVersion": "2015-06-15"
    "location": "[resourceGroup().location]",
    "publicIPAddressName": "myPublicIP",
    "publicIPAddressType": "Dynamic"
  },
  "resources" : [
    {{ public_ip_address.from_vars() }}
  ]
}
