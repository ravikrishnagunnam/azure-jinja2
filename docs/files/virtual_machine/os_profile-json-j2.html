create

Create a new os profile to be used in a virtual machine (see virtual_machine).

Parameters

namestring, name for the computer
adminUsernamestring, name for the admin user
adminPasswordstring, password associated with the admin user
sshKeyDatastring, public key to login if required
sshKeyPathstring, full filename of the authorized_keys file
customDatastring (optional), base64-encoded custom data

Example 1

{% import 'azure-jinja2/macros/virtual_machine/os_profile.json.j2' as vm_os_profile %}

{% set OS_profile %}
{{ vm_os_profile.create(
      name = "myserver",
      adminUsername = "azureuser",
      adminPassword = "myp@ssw0rd!",
   )
}}
{% endset %}
