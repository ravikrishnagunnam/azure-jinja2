Macros

create

Create a virtual network (VNET).

Parameters

namestring
apiVersionstring
locationstring
addressPrefixstring, virtual network range in CIDR notation (ex: “10.0.0.0/16”)
subnetNamestring
subnetPrefixstring, subnet range in CIDR notation (ex: “10.0.0.0/24”), must be included inside addressPrefix

Example

{% import 'azure-jinja2/macros/virtual_network.json.j2' as virtual_network %}

{
  "$schema": "https://schema.management.azure.com/schemas/2015-01-01/deploymentTemplate.json#",
  "contentVersion": "1.0.0.0",
  "resources" : [
    {{ virtual_network.create(
         name = "myVNET",
         apiVersion = "2015-06-15",
         location = "westeurope",
         addressPrefix = "10.0.0.0/16",
         subnetName = "mySubnet",
         subnetPrefix = "10.0.0.0/24"
       )
    }}
  ]
}

from_vars

Create a virtual network (VNET) from variables.

Parameters

namestring = “[variables(‘virtualNetworkName’)]”
apiVersionstring = “[variables(‘apiVersion’)]”
locationstring = “[variables(‘location’)]”
addressPrefixstring = “[variables(‘addressPrefix’)]”, virtual network range in CIDR notation (ex: “10.0.0.0/16”)
subnetNamestring = “[variables(‘subnetName’)]”
subnetPrefixstring = “[variables(‘subnetPrefix’)]”, subnet range in CIDR notation (ex: “10.0.0.0/24”), must be included inside addressPrefix

Example

{% import 'azure-jinja2/macros/virtual_network.json.j2' as virtual_network %}

{
  "$schema": "https://schema.management.azure.com/schemas/2015-01-01/deploymentTemplate.json#",
  "contentVersion": "1.0.0.0",
  "variables": {
    "apiVersion": "2015-06-15"
    "location": "[resourceGroup().location]",
    "virtualNetworkName": "MyVNET",
    "addressPrefix": "10.0.0.0/16",
    "subnetName": "Subnet",
    "subnetPrefix": "10.0.0.0/24",
  },

  "resources" : [
    {{ virtual_network.ifrom_vars() }}
  ]
}
