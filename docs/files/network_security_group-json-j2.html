Macros

create

Create a new network security group.

Parameters

namestring
apiVersionstring
locationstring
securityRulesarray, list of security rules to enforce (see network_security_group/security_rules)

Example

{% import 'azure-jinja2/macros/security_rules.json.j2' as security_rules %}
{% import 'azure-jinja2/macros/network_security_group.json.j2' as network_security_group %}

   "resources" : [
       ...,
       {% set jumbox_security_rules %}
       [ {{ security_rules.SSH() }}, {{ security_rules.HTTPS() }} ]
       {% endset %}
       {{ network_security_group.from_vars(securityRules = jumbox_security_rules) }}
   ]

from_vars

Create a new network security group from JSON variables.

Parameters

namestring = “[variables(‘networkSecurityGroupName’)]”
apiVersionstring = “[variables(‘apiVersion’)]”
locationstring = “[variables(‘location’)]”
securityRulesarray = “[]”, list of security rules to enforce (see <security_rules>)

Example

{% import 'azure-jinja2/macros/security_rules.json.j2' as security_rules %}
{% import 'azure-jinja2/macros/network_security_group.json.j2' as network_security_group %}

   "resources" : [
       ...,
       {% set jumbox_security_rules %}
       [ {{ security_rules.SSH() }}, {{ security_rules.HTTPS() }} ]
       {% endset %}
       {{ network_security_group.from_vars(securityRules = jumbox_security_rules) }}
   ]
